# Chroniq V2 - Video Engagement Improvements

## Goal

This pass makes generated videos feel more edited and less like image slideshows by adding style presets and editorial scene metadata.

## What Changed

- Added render style presets:
  - `business_documentary_dark`
  - `tech_history_fast`
- Added `VIDEO_STYLE` to `.env.example`.
- Extended scene planning to request editorial metadata:
  - `sceneType`: `image`, `headline`, `stat`, `quote`, `timeline`, or `comparison`
  - `headline`: short on-screen label
  - `emphasis`: strongest word, number, year, or company to highlight
  - `motion`: Ken Burns motion direction
- Saved the new scene metadata into `scene_manifest`.
- Passed the metadata into Remotion during render.
- Added Remotion editorial overlays for headline/stat/quote/timeline/comparison scenes.
- Connected captions and vignette styling to the active style preset.

## Usage

```ini
VIDEO_STYLE=business_documentary_dark
```

Use `business_documentary_dark` for case studies, business failures, startup stories, and dramatic explainers. Use `tech_history_fast` for faster technology history or product evolution shorts.

## Notes

- No database migration is required because scene details are stored in the existing JSON `scene_manifest`.
- Existing videos without the new scene fields still render normally.
- Dashboard/API behavior is unchanged in this version.
